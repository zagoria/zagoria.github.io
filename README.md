# 道德經 · Dào Dé Jīng

A minimal, elegant web reader for the *Tao Te Ching* — readable as a Progressive Web App.

**[→ Read it live](https://zagoria.github.io/daodejing/)**

---

## Features

- All 81 chapters with Chinese original and multiple translations
- Translations by James Legge (1891) and Chu Ta-Kao (1904)
- Installable as a PWA (works offline, no app store needed)
- Distraction-free reading with a paper-textured aesthetic

## Translations included

| Translator | Year |
|---|---|
| James Legge | 1891 |
| Chu Ta-Kao | 1904 |

## Install as app

On mobile: open the live link in your browser, then tap **"Add to Home Screen"**.  
On desktop: click the install icon in the address bar.

## Files

| File | Purpose |
|---|---|
| `index.html` | Main app |
| `manifest.json` | PWA manifest |
| `playstore-icon.html` | Icon source (render to generate PNG icons) |

## Local development

No build step needed — open `index.html` in a browser.
